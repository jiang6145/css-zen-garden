# CSS Zen Garden 切版

## 介紹

切版練習，HTML 結構不可變動，只使用 CSS 去改變 CSS Zen Garden 網頁樣式。
我以深色系搭配鮮豔的花為主題風格。

## 使用技術

- CSS / SCSS
- jQuery
